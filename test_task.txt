Техническое задание на позицию Python (Django) back end developer
Нужно реализовать иерархическую структуру товаров, где предпоследний уровень будет связан друг с другом, пример здесь:
https://metalvis.ua/uk/catalog/prod-10110-837510640012-shurup-universalnyi-s-potainoi-golovkoi-s-krestoobraznym-shlicem/

Схема БД:
Категория товара (1я модель)
Название
Описание
Есть 4 уровня вложенности категорий: section -> group -> subgroup -> card (Карточка товара, на которую ссылается продукт)

Товар (2я модель)
Название
Описание
Сфера применения
Диаметр (float)
Длина (int)
Цвет
Изображение


Эндпоинты:
Список категорий
Список всех карточек товаров категории (секции, группы, подгруппы)
Детальная информация о карточке
Список продуктов по определенной карточке

В карточке товаров должна выводится следующая информация в виде одного объекта:
Название
Сферы применения (список уникальных значений, который берется из продуктов, которые ссылаются на данную карточку)
Диаметр (список уникальных значений, который берется из продуктов, которые ссылаются на данную карточку)
Длина (список уникальных значений, который берется из продуктов, которые ссылаются на данную карточку)
Цвет (список уникальных значений, который берется из продуктов, которые ссылаются на данную карточку)
Изображение главное
Изображения по цветам
